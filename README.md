### 方式二：源码安装

要求 Go 1.6+

1、下载 gvt 依赖管理工具
	
	go get github.com/polaris1119/gvt
	
下载后将 gvt 加入 PATH 中。

2、下载安装依赖

cd 到 studygolang 源码目录	

	// unix
	./getpkg.sh
	// windows
	getpkg.bat

3、编译 studygolang
	
	// unix
	./install.sh
	// windows
	install.bat

这样便编译好了 studygolang

## 步骤三

在 studygolang 源码中的 bin 目录下应该有了 studygolang 可执行文件。

接下来启动 studygolang。

	// unix
	./start.sh
	// windows
	start.bat

或者
	
	// unix
	bin/studygolang
	// windows
	bin\studygolang.exe

一切顺利的话，studygolang 应该就启动了。
